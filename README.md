# Python Voice Assistant (Jarvis Inspired)

An AI-powered voice assistant built with Python that can listen to voice commands, process them intelligently using OpenAI, and respond with natural speech.

---

## Features

- Voice Recognition (SpeechRecognition + PyAudio)
- AI-powered responses (OpenAI API)
- Text-to-Speech (pyttsx3 / gTTS)
- Audio playback support (pygame)
- Modular and extendable structure

---

## Tech Stack

- Python
- OpenAI API
- SpeechRecognition
- PyAudio
- pyttsx3
- pygame
- requests

---

## Installation

### 1) Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/python-voice-assistant.git
cd python-voice-assistant
```

### 2) Create virtual environment (Recommended)

```bash
python -m venv venv
source venv/Scripts/activate   # Windows
# source venv/bin/activate     # Mac/Linux
```

### 3) Install dependencies

```bash
pip install -r requirements.txt
```

## Run the Assistant

```bash
python main.py
```
